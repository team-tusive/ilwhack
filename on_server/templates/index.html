Aggregates and parses details from Lothian and Borders Police Reports, enabling the utilization and leveraging of crime statistics to create actionable turnkey logistics/transportation solutions for end-users and thereby cultivate user empowerment and maximize user travel confidence. Also allows the user to dynamically explore crime statistics via an interactive data visualization interface.
